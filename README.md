# 📚 Libronexa

A modern library management platform built with React, TypeScript, and Tailwind CSS featuring dynamic UI, book tracking, borrow management, and an intuitive admin dashboard.

---

## 🚀 Features

- 📖 Browse and explore books
- 🔍 Smart search functionality
- 📦 Borrow request system
- 👨‍💼 Admin dashboard (manage books & users)
- 🌙 Dark mode UI
- 🎨 Modern animations (Framer Motion)
- 📱 Fully responsive design

---

## 🛠️ Tech Stack

- React (TypeScript)
- Tailwind CSS
- Framer Motion
- React Router

---
